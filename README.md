# K-drama-Recommendation
The K-Drama Recommendation System is a simple Python-based application that helps users discover Korean dramas based on their preferences. It reduces the time spent searching for shows by providing quick and relevant recommendations.


Overview
The K-Drama Recommendation System is a simple Python-based application that helps users 
discover Korean dramas based on their preferences. It reduces the time spent searching for 
shows by providing quick and relevant recommendations.


Problem Statement
With the increasing popularity of Korean dramas, users often struggle to choose what to 
watch due to the vast number of options available. This project aims to simplify the decisionmaking process by recommending dramas based on user input.
Features
• Recommend K-dramas based on genre
• Display top-rated dramas
• Simple and user-friendly interface
• Fast and efficient filtering


Tech Stack
• Language: Python
• Data Handling: Lists / CSV file
• Platform: Local system


Project Structure
kdrama-recommendation/
│── kdrama.py
│── README.md


How to Run the Project
1. Make sure Python is installed
2. Download or clone this repository
3. Open terminal in the project folder
4. Run the program:
python kdrama.py


How It Works
• The program stores a list of K-dramas with their genres and ratings
• The user selects a preferred genre
• The system filters and displays matching dramas
• Results are shown based on relevance or rating


Sample Output
Enter preferred genre: Thriller
Recommended K-Dramas:
1. Crash Landing on You
2. Descendants of the Sun
3. Goblin


Limitations
• Uses a small dataset
• Recommendations are not personalized
• No graphical user interface
